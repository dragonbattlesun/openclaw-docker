## 第 15 课配图直读补充

以下页码来自 `memory/chanlun_pdf_figures/pages/page_064-064.png` 及 `page_066-066.png` 到 `page_071-071.png`,用于背驰和 MACD 辅助判断时回查:

- `page_064`: 30m 图上 MACD 红柱三波一波比一波低,是明显背驰信号;短线资金可用 1m/5m 找第一类买点,但大资金不宜按 1m/5m 全仓进出。
- `page_066`: 判断 MACD 背驰不能只看当天红柱是否比前一天短;要看一整波红/绿柱从伸长到缩短的转折,以及该转折高度是否低于上一波。
- `page_067`: 单看红/绿柱不够,还要看黄白线 `DIF/DEA` 走势。最有效的顶背驰常见于黄白线回到 0 轴附近再上,价格创新高而黄白线和柱峰不创新高。
- `page_068`: 30m 背驰可能早已发生,但日线未必存在背驰;买入若不是日线/周线第一、第二类买点,后续就应按实际买入级别和箱体边界处理。
- `page_069`: MACD 定律: 第一类买点通常在 0 轴下背驰形成,第二类买点通常在第一次上 0 轴后回抽确认形成,卖点反过来。不要把图看得太散,要一大段地看。
- `page_070`: 580991 与 038004 案例强调要自己复盘 MACD 定律;三角形整理不看均线,直接看图形高低点收敛。
- `page_071`: 038004 展示三角形整理构造第二类买点;缺口不必一概回补,突破性或岛型反转缺口不补是强势,盘整上涨末端衰竭缺口才更需警惕。

## 第 16 课配图直读补充

以下页码来自 `memory/chanlun_pdf_figures/pages/page_072-072.png` 到 `page_075-075.png`,用于低位启动选股和中小资金买卖法回查:

- `page_072`: 任意同级别走势可拆成上涨、下跌、盘整三类;六种组合里,有买入价值的是 `下跌+上涨`、`下跌+盘整+上涨`、`上涨+盘整+上涨`。但用于选股时,要先在 `下跌+盘整+下跌` 中找第二段下跌的第一类买点候选。
- `page_073`: 中小资金不参与盘整;买入后若出现盘整走势,无论后面如何先退出。最有效方法是只参与可能演化成 `下跌+上涨` 的结构。
- `page_074`: 标准选股程序:先选 `下跌+盘整+下跌`,再在第二段下跌出现第一类买点时介入,介入后一旦出现盘整走势坚决退出。
- `page_075`: 若第二段上涨出现第一类卖点,通常要走;这对应后面可能演化成 `上涨+下跌` 的典型走势。中小资金应减少持股数量,集中跟踪高质量结构。

## 第 20 课配图直读补充

以下页码来自 `memory/chanlun_pdf_figures/pages/page_099-099.png` 到 `page_105-105.png`,用于中枢级别扩张、延伸和第三类买卖点判断时回查:

- `page_099`: 开篇明确中枢级别扩张的前提: 三个连续次级别走势类型的重叠确定中枢;若前后三段与原中枢延续没有必要重叠,不能直接定性为原级别趋势,而是可能生成更大级别中枢。
- `page_100`: 两个延续定理: 更高级别中枢产生前,原级别走势类型只能延续;更高级别中枢产生,当且仅当围绕连续两个同级别中枢产生的波动区间重叠。三段中枢区间通式为 `[max(a2,b2,c2), min(a1,b1,c1)]`,图例可简化为 `[max(a2,c2), min(a1,c1)]`。
- `page_101`: 中枢中心定理一: 后续任意 `[dn, gn]` 与 `[ZD, ZG]` 有重叠就是原中枢延伸;若 `dn > ZG` 或 `gn < ZD`,必然进入高级别中枢生成或走势延续分类。
- `page_102`: 中枢中心定理二: 前后同级别中枢的 `GG/DD/ZG/ZD` 关系决定上涨、下跌、延续或生成高一级别中枢。三买/三卖来自离开中枢后的首次有效回试/回抽。
- `page_103`: 图示归纳回升中枢、回调中枢、中枢延伸、前后中枢关系和形成高级别中枢的基本形态。实战中先定中枢边界,再判断离开/返回。
- `page_104`: 第三类买卖点必须是第一次回试/回抽。三买后三买本身不保证趋势,仍可能进入更大级别盘整;若不能趋势化,按盘整高点处理。
- `page_105`: 不要把中枢自身变化与围绕中枢的离开/返回混在一起。底部图形要先确认级别,短线回调可看小级别图。

## 第 24 课配图直读补充

以下页码来自 `memory/chanlun_pdf_figures/pages/page_134-134.png` 到 `page_143-143.png`,用于 MACD 辅助背驰和盘整背驰判断时回查:

- `page_134`: 背驰与买卖点有级别递归关系。任一背驰都必然制造某级别买卖点,任一级别买点都来源于某级别走势背驰;但转折不等于永久反转,必须回到对应级别走势继续判断。
- `page_135`: 标准 MACD 背驰辅助必须在 A+B+C 趋势结构中使用。B 段通常把黄白线拉回 0 轴附近,C 段 MACD 面积小于 A 段才构成辅助背驰证据。
- `page_136`: 盘整背驰中,若 C 段未突破中枢且面积小于 A 段,后续按回拉/回跌处理;若 C 段突破中枢但力度不足,先兑现,再看回拉是否回中枢来区分三买还是盘整延续。
- `page_137`: 背驰或盘整背驰后,回拉至少要观察是否回到 B 段中枢。MACD 只是辅助,严格结构仍以中枢和走势类型为准。
- `page_138`: 三角形常是一段走势最后一个调整中出现的形态。三角形后若继续上攻,要盯 30m/60m 或操作级别背驰,可先打短差风控。
- `page_139`: 快速赶顶股票的最后上涨可能只是 1m 趋势延伸;1m 标准背驰也可能快速打回延伸起点附近,不能因为级别低忽视短线风险。
- `page_140`: 先分清日线、30m 和 5m 所处走势。上级别明确上涨时,5m 背驰回撤通常有限;走势是各方合力结果,不要把庄家力量从走势中单独拿出来解释。
- `page_141`: 600832 案例说明,日线三买后的复杂回拉可以是三段或五段三角形,C 段不一定创新低,三买结束点不一定是整个回拉最低点。最后一次回拉不回原中枢才是关键。
- `page_142`: 黄白线创新高时通常不是本级别背驰,可能只是 1m 背驰。日线有缺口时低级别自然有缺口,低级别软件看不到只是软件口径问题。
- `page_143`: 600217 案例说明,判断 MACD 背驰要先让黄白线回抽 0 轴,随后价格创新高而黄白线不创新高、柱子面积变小。三角形最佳介入点仍在下边附近。

## 第 25 课配图直读补充

以下页码来自 `memory/chanlun_pdf_figures/pages/page_144-144.png` 到 `page_159-159.png`,用于均线“吻”、MACD 中枢提示、父级别中枢构成和操作级别纪律回查:

- `page_144`: “吻”、MACD、背驰和中枢是不同系统。601333 案例强调日线中枢由回试中的次级别走势构成,是否摆脱中枢要继续看后续走势,不影响按当下结构操作。
- `page_145`: 不要混淆均线中枢和缠论中枢。走势必完美意味着必有两次拉回,第二次拉回不一定低于第一次。第三类买点只能在中枢之外,三角形最后买点只能是次级别买点。
- `page_146`: 阴线买入必须是下跌或回试中形成买点,不能追高后再找理由。下跌中枢对后续上涨只是可能阻力参考。中枢只能是回试时形成的。
- `page_147`: 日线中枢需要 30m 三段走势类型;30m 中枢需要至少三段 5m 走势类型。不能用 30m 一条直线走势确认日线中枢。
- `page_148`: 背驰要看结构、黄白线和柱子,不能只看柱子。没有黄白线对 0 轴的回拉,本级别通常不存在标准背驰,可能只是小级别背驰。
- `page_149-150`: 大盘中线可能形成周线走势中枢,牛市第一波未完成。个股机会与大盘中枢震荡可并存,但技术上仍按背驰和三角形调整节奏处理。
- `page_151`: 操作级别映射: 周线偏中线,日线偏短线,30m 短线,5m 超短线,1m 只对 T+0 或极短短差有意义。
- `page_154-156`: 60m 背驰需要用 60m 级别确认。日线不需要看 1m 做结论,只有次级别背驰或精确定位时下钻。快速拉开末段中,1m 背驰也可能引发剧烈回落,但仍先按低级别风险处理。
- `page_157-159`: 短差可以先卖后买或先买后卖,但资金和股票要绑定固定级别不断流动。技术分析只制定当下计划,不预测不可确定的未来。
- `page_160-161`: 盘整中围绕中枢做短差,操作上不能空等“确认”。卖点卖、买点买,按各自级别处理;中线仓可用 5m/30m 卖点先卖部分,再等买点回补。
- `page_162-163`: 一线股、权重股常用于盘整控盘和突破,短线机会更多在 5m/30m;二三线股更重发展。工程上这属于选股/资金层,不能改变买卖点结构。
- `page_164`: 第二类买点必须等待次次级别背驰或双次回拉确认;否则只能是二买候选。震荡中要敢于来回动,但必须有卖点和买点依据。
- `page_165-166`: 1m 背驰只适合急剧变动中的机动资金。标准 1m 底背驰示例: 黄白线回抽 0 轴后再破低位,价格创新低而 MACD 不创新低。若 5m 无背驰,不能把 1m 背驰包装成 5m/30m 结论。
- `page_167`: 第一类买点通常低于第二类买点,第二类低于第三类;卖点反过来。MACD 比简单上下位更直观,但仍不能替代走势结构。

- `page_820`: 第二类买点是第一类买点后直接次级别回抽结束点。最强是二买与三买合一,常对应 V 型快速回升;最弱是二买跌破一买,通常对应盘整背驰和平台扩张;一般情形是一、二、三买依次抬高。中阴状态结束后,不再有第一/第二类买卖点,只看第三类买卖点和中枢震荡买卖点。

## 后续分析校验清单

个股缠论分析遇到争议时,按以下顺序复核:

1. 分型 / 笔 / 线段争议: 优先查第 62、65、67、78、82-84 课配图。
2. 中枢争议: 优先查第 20 课配图,确认是否来自三个连续次级别走势类型的重叠。
3. 三买 / 三卖争议: 查第 20、53、107 课,确认是否满足离开中枢后的第一次有效回试 / 回抽。
4. 背驰争议: 查第 15、24、37、44 课,确认是否有趋势、是否有同级别可比段、是否只是盘整背驰。
5. 级别争议: 查第 38、39、40、84、89、91、102 课,确认操作级别没有临时切换、没有因时间自动升级,也没有把子级别中阴/笔状态噪音当父级别信号。
6. 板块与过滤争议: 查第 106 课。板块强弱只决定优先级和仓位,不能替代买卖点。

## 对本地工程的约束

- `chanlun_native.py` 可继续作为候选结构生成器,但输出需要标注 `候选`。
- 严格三买不能只用三笔重叠 + 突破判定,必须检验离开、首次回试、不破 `ZG`。
- 标准一买不能只用低位 + 底分型判定,必须有本级别下跌趋势背驰背景。
- 低位启动模型的 `1B/2B/3B` 是工程标签,用户输出中必须区分 `工具标签` 与 `原文结构`。
- 个股决策仍执行日线 + 60m + 30m 三级共振,周线 / 月线只做方向定位。
