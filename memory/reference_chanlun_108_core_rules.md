## 置顶哲学原则: 市场哲学的数学原理

更新时间: 2026-06-05。原因: 用户要求将“缠师用数学几何函数分析 K 线自同构性,其哲学本质是人的贪嗔痴疑慢引发自相似性和走势级别自组织”置顶为后续缠论分析的总原则。

## 适用场景

- 解释缠论是什么、为什么不同级别走势会相似、为什么小级别能递归生成大级别。
- 个股缠论分析、买卖点判断、涨停回溯、持仓诊断和做 T 计划中,需要决定“能否用小级别信号参与大级别走势”时。
- 遇到突然拉涨停、无回踩直线上涨、工具标签和人工结构冲突时,用于防止事后硬解释买点。

## 核心定义

缠论的哲学本质是: 市场由人的贪、嗔、痴、疑、慢等群体行为驱动,这些行为在不同时间尺度上反复出现,形成走势的自相似性;缠论用数学几何化的递归结构,把这种自相似性表达为走势自同构和级别自组织。

工程化表达:

```text
人性重复 -> 价格轨迹自相似 -> 分型/笔/线段/中枢递归同构 -> 走势级别自组织 -> 买卖点可证伪
```

这不是“相似图形必然重复”的预测术,而是“任一级别都使用同一套结构规则”的描述和执行框架。任何买卖点都必须回到本操作级别的结构条件、触发位和失效位。

## 判定步骤

1. 先固定操作级别,说明本次判断是日线、60m、30m、5m 还是更小级别。
2. 在该级别内独立完成 `分型 -> 笔 -> 线段 -> 中枢 -> 走势类型 -> 买卖点` 判断。
3. 下钻次级别时,只用于精确定位、背驰验证、三买三卖回试验证、止损或卖点确认。
4. 判断能否递归升级: 小级别信号必须先生成本级别反向笔/走势类型,再形成本级别中枢,再离开中枢,再回试不回中枢;缺一步只能写候选。
5. 如果直接拉涨停或急速离开,没有本级别回试,则只写“离开段 / 突破确认 / 持仓确认 / 小级别试错窗口”,不能写成本级别严格 3B。
6. 如果本级别和小级别都没有低风险参与点,必须写“没有清晰买点,错过就错过”,不能因事后上涨反推买点。

## 输出要求

必须写清:

- `哲学口径`: 人性自相似 -> 走势自同构 -> 级别自组织。
- `本级别结论`: 当前是中枢内、离开中枢、回试中、买点候选还是无买点。
- `次级别支持/反证`: 小级别是否只是试错窗口,能否升级,缺哪一步。
- `买点标签`: 工具标签 / 人工结构候选 / 原文严格结论 / 当前执行。
- `执行纪律`: 仓位、A 股 T+1、失效位、是否因止损距离过大而放弃。

禁止写法:

- “直接上涨没有回踩,所以缠论失效。”
- “5m 有买点,所以日线买点成立。”
- “涨停突破中枢,所以日线 3B 已成立。”
- “小级别也没有清晰参与点,但后面涨了,所以 09:35/09:45 应该算买点。”

## 正例

- `日线离开中枢,但没有回试`: 写“日线离开段成立,日线严格 3B 未成立;已有仓可按小级别卖点管理,空仓等回试或下钻找小级别结构完整买点”。
- `5m 出现止跌反转,但止损距离过大`: 写“5m 试错窗口,非清晰低风险买点;A 股 T+1 下只能 loose 2-3% 或放弃”。
- `30m 突破确认,但无回踩`: 写“30m 突破确认 / 持仓确认,不是 30m 严格 3B”。

## 反例

- 把日线中枢上方的涨停直接命名为日线三买,错在缺少第一次次级别回试不回中枢。
- 把 5m 急拉长阳命名为明确买点,错在买点和失效位距离过远,不符合低风险参与。
- 看到相似形态就预测主升,错在把自同构理解成图形宿命,没有落实到级别、中枢、触发位和失效位。

## 优先级

本原则是解释缠论和约束输出的置顶哲学口径,但不替代原文严格买卖点定义。若与工具标签、临时扫描信号或旧提示词冲突,以本原则和原文结构定义为准;若与“三买必须第一次回试不回中枢”等严格定义冲突,严格定义优先。

## 教训案例

2026-06-04 三安光电 `600703`: 6月3日只能列为观察候选;6月4日直接拉涨停,30m 没有严格买点,5m 只有止损距离偏大的试错窗口。正确复盘是“日线离开段 + 持仓确认 + 空仓无高质量参与点”,不能事后把突然涨停包装成标准 30m/日线买点。

## 级别与显微镜

- 级别不是 K 线周期本身。1m/5m/30m/日线只是观察窗口,级别来自走势结构递归。
- 选定操作级别后,应把已经完成的次级别走势类型当作本级别的一段来看。
- 下钻小级别只用于三种情况: 背驰段精确定位、三买三卖确认、止损/卖点确认。
- 不允许在亏损、持仓安慰或预测需要下临时切换级别。

### 走势自同构性

缠论走势自同构性不是简单的图形相似,而是结构递归同构:任一级别都服从 `分型 → 笔 → 线段 → 中枢 → 走势类型 → 买卖点` 这套规则。

- 大级别的一笔 / 一段,内部可以由次级别完整走势类型递归生成。
- 大级别中枢,内部可以包含多个小级别中枢;但中枢归属必须按本级别的次级别走势重叠来确认。
- 小级别买卖点只能辅助本级别定位和风控,不能直接替代本级别买卖点定性。
- `5m 底背驰` 只能写成 `30m 转折候选 / 下沿回补候选`,不能直接写成 `30m 严格1B`。
- `30m 三买` 只能说明 30m 级别趋势确认候选,不能直接升级为日线三买;日线三买必须由日线级别中枢离开和回试关系确认。
- 小转大升级链条必须完整:小级别背驰候选 → 本级别反向笔 / 走势类型完成 → 本级别中枢形成 → 离开中枢 → 第一次回踩不回中枢。缺一步只能写候选。
- 输出必须写清:本级别结论、次级别支持/反证、能否递归升级、升级触发位、升级失败位。

## 分型、笔、线段

- 分型、笔、线段定义适用于任何周期 K 线图;先看高低点,不按阴阳线颜色下结论。
- 顶分型: 相邻三根 K 线中,第二根高点最高且低点也最高;底分型: 第二根低点最低且高点也最低。
- 顶分型的最高点才是该分型的顶,底分型的最低点才是该分型的底;“顶分型的底”“底分型的顶”没有结构意义。
- 顶分型/底分型必须先处理 K 线包含关系。相邻两根有包含时,向上取较高高点和较高低点,向下取较低低点和较低高点,递归合并成无包含序列后再判分型。
- 一笔必须由相邻顶底构成,且顶底之间至少有一根独立 K 线;顶底共用同一根 K 或中间无其他 K 线时,不能强行算一笔。
- 两个顶或两个底之间若夹着其他顶/底,只是把多根笔当成一笔,必须继续拆;若中间没有其他顶/底,前一个顶/底后的转折级别太小,可忽略不计。
- 线段至少三笔,且前三笔必须有重叠部分;不是任意连续三笔都能构成线段。
- 线段破坏必须由另一个线段完成。等价说,线段被破坏当且仅当至少被有重叠部分的连续三笔中的一笔破坏;单纯一笔不能确认线段结束。
- 笔、线段用于确认具有操作意义的最低级别中枢和走势类型。严格说,线段内部没有标准中枢/背驰;只能标注为 `类中枢`、`类背驰力度比较`。
- 线段划分必须转成特征序列: 向上笔开始的线段取其中所有向下笔 `X`,向下笔开始的线段取其中所有向上笔 `S`;相邻特征序列元素无重合区间就是 `缺口`。
- 特征序列也要先处理包含关系,得到标准特征序列后再判分型。向上笔开始的线段只考察特征序列顶分型;向下笔开始的线段只考察特征序列底分型。
- 第一种情况: 特征序列顶/底分型的第一、第二元素无缺口,线段在该顶/底分型高/低点结束。第二种情况: 第一、第二元素有缺口,必须等待反向笔序列的特征序列出现对应底/顶分型后,才能确认原线段结束。
- 若未出现标准特征序列分型,或第二种情况的后续分型未确认,不得提前把走势切成 A/B/C 多个线段;优先维持原线段延伸或待确认。
- 特征序列包含处理只能在同一线段的特征序列元素之间进行,不能用视觉上的跨序列包含替代规则。
- 后续个股分析中的人工合笔,必须列出每笔方向、起讫、区间和 K 数,并标注是否满足独立 K 与顶底不重叠。

## 中枢

- 中枢由至少三个连续次级别走势类型的重叠部分构成。
- 常用工程表达: `ZD = max(低点序列)`, `ZG = min(高点序列)`, 但必须说明这些段来自哪个级别的次级别走势。
- 中枢震荡没有本级别一买/二买/三买,只存在更小级别买卖点。
- 中枢震荡中不要追涨杀跌;向上力度衰竭卖,向下力度衰竭回补。
- 中枢延伸等价于后续任意区间 `[dn, gn]` 与原 `[ZD, ZG]` 有重叠。若 `dn > ZG` 或 `gn < ZD`,就不再是原中枢延伸,必须归入更高级别中枢生成或原级别趋势延续。
- 前后同级别两个中枢的关系决定走势方向: 后 `GG < 前 DD` 等价于下跌及延续;后 `DD > 前 GG` 等价于上涨及延续。边界交错时可能形成更高级别中枢。

## 三类买卖点

- 一买: 本级别下跌趋势背驰点。没有趋势,没有标准一买。
- 二买: 一买后第一次次级别回调不破前低或盘整背驰确认;小转大场景中,二买往往比一买更实用。
- 三买: 次级别走势向上离开本级别中枢,再以次级别走势回试,低点不跌破 `ZG`。
- 三卖: 次级别走势向下离开本级别中枢,再以次级别走势回抽,高点不升破 `ZD`。
- 第三类买卖点必须是离开中枢后的第一次有效回试/回抽。后续多次不回中枢,不能重复标为本级别三买/三卖。
- 三买回试可以是三段式或五段式复杂回拉,C 段不一定创新低,结束点也不一定是整个回拉最低点。复杂三角形回拉中,只要最后一次回拉不回原中枢,即可进入三买确认流程。
- 若第一次回抽回到中枢,原离开失败,不能叫三买;后续若重新离开中枢,再出现首次回抽不回中枢,可按新的结构关系判断三买。
- 第三类买点后不必然走趋势,可能转成更大级别盘整;若后续不能形成趋势,应在盘整高点处理。
- 第三类买卖点可与不同级别的第二类买卖点重合,但必须分别标明级别。不能把小级别二买和大级别三买混写成一个无级别标签。
- 三买后若不能形成趋势,仍按盘整高点卖出;三卖后若不能延续下跌,也要按回补点处理。

## 左侧买点边界

更新时间: 2026-06-16。原因: 固化“缠论允许左侧,但只允许结构完成后的左侧; 不允许下跌途中猜底式左侧”的理论边界。

### 适用场景

- 个股缠论分析、持仓诊断、做 T、买点标签、扫描器候选解释中,凡涉及 `一买 / 1B候选 / 盘整背驰 / 中枢下沿回补 / 左侧试错` 都必须使用本规则。
- 用户问“能不能左侧买 / 是不是一买 / 能否低吸 / 下沿能不能回补”时,必须先按本规则定性,再谈执行。

### 核心定义

- 缠论允许左侧,但只允许 `结构完成后的左侧`: 本级别下跌走势类型完成、C 段相对 A 段创新低、同级别背驰或盘整背驰力度衰竭成立、反弹笔成立,且有明确失效位。
- 缠论不允许 `下跌途中猜底式左侧`: 只有跌幅大、价格低、指标低、故事好、接近历史低点、或单根 K 线止跌,但本级别结构未完成时,不能叫一买或当前买点。
- 中枢震荡里的下沿回补、盘整背驰、类背驰力度衰竭有左侧交易味道,但不能升级为本级别严格一买;只能写成 `盘整背驰候选 / 中枢下沿回补 / 小级别买点 / 执行试错仓`。

### 判定步骤

1. 定操作级别: 日线、60m、30m、5m 必须分开判断,不能混级别拼买点。
2. 合笔与线段: 列出最近笔方向、起讫、区间、K 数和是否完成;未完成笔不能确认买点。
3. 判断走势类型: 若本级别仍是中枢震荡,直接写 `本级别无严格 1B/2B/3B`。
4. 严格一买: 必须满足本级别 `a+A+b+B+c` 下跌趋势、A/B 两个同级别中枢、后 `GG < 前 DD`、C 段创新低、同级别 MACD 面积背驰、反弹笔成立。
5. 盘整背驰/下沿回补: 只可标为候选或小级别买点;若回拉不回原中枢,后续再观察是否形成三买;若回到中枢,按中枢震荡延续处理。
6. 执行约束: 所有左侧候选必须靠近失效位;A 股 T+1 下今日新买股份不能当天卖,次一交易日才可执行止损。

### 输出要求

必须拆成四层写:

```text
工具标签:
人工结构: 严格一买 / 一买候选 / 盘整背驰候选 / 中枢下沿回补 / 小级别买点
原文严格: 严格1B已成立 / 原文严格1B未成立
当前执行: 等待 / 执行试错仓 / 不试错
失效位:
```

禁止只写“左侧可以买”“有一买”“跌多了可低吸”。必须写清级别、结构是否完成、MACD 面积和柱峰/DIF护栏、量能是否反证、T+1 和失效位。

### 正例

- `30m 本级别 a+A+b+B+c 下跌趋势成立,C 段创新低且绿柱面积小于 A 段,反弹 Up 笔完成。人工结构: 严格1B候选; 原文严格: 严格1B候选; 失效位: C 段低点; 当前执行: 只可按左侧试错仓或等待2B。`
- `30m 仍在中枢震荡,下沿出现 5m 盘整背驰。人工结构: 中枢下沿回补候选/小级别买点; 原文严格: 30m 严格1B未成立; 当前执行: 靠近失效位才允许 loose 试错。`

### 反例

- `跌到历史低位,所以是日线一买`。错在没有本级别趋势、C 段、背驰、反弹笔和失效位。
- `中枢下沿反弹就是本级别一买`。错在中枢震荡没有本级别严格一买,只能写盘整背驰候选或小级别回补。
- `5m 底背驰就是 30m 严格1B`。错在小级别信号不能直接替代本级别买点。

### 优先级

- 本规则优先于任何“低位启动 / 超跌反弹 / 下沿低吸 / 工程 1B 标签”的宽松说法。
- 若与旧输出习惯冲突,按本规则改写: 未达原文严格一买时,只能写候选、观察、等待 2B/3B 或 `执行试错仓`,不能写确认仓、趋势仓或本级别严格一买。

## 背驰与盘整背驰

- 没有趋势,没有背驰。盘整里只能说盘整背驰或力度衰竭。
- 标准背驰要求同级别趋势结构中有两段同向走势可比,常见形式为 `a + A + b + B + c`, 且 A/B 为同级别中枢。
- 上涨背驰必须在 c 段创新高后比较力度;下跌背驰必须在 c 段创新低后比较力度。
- MACD 辅助口径: B 段通常把黄白线拉回 0 轴附近;比较 A 段与 C 段红/绿柱面积。
- DIF/DEA 回抽或停留在 0 轴附近,只能提示至少当前级别可能形成中枢;不能替代价格结构上的三段次级别走势类型重叠,也不能直接确认更高级别中枢。
- 均线系统的“吻”只能辅助观察走势节奏,不是缠论中枢。中枢仍以连续三段次级别走势类型重叠为准。
- 盘整背驰若离开中枢后回不回中枢,决定是继续中枢震荡还是演化为第三类买卖点。
- 标准 MACD 背驰辅助必须放在 A+B+C 趋势结构里:A 前已有中枢,B 是另一中枢并通常把黄白线拉回 0 轴附近,C 段完成时 MACD 面积小于 A 段。
- 盘整背驰中,若 C 段突破中枢但力度小于 A 段,原则上先兑现;随后若回拉不回中枢,可按次级别一买回补并观察三买;若回到中枢,盘整延续。
- 背驰或盘整背驰后的回拉,至少要观察是否回到 B 段中枢。不回中枢看三买/三卖,回到中枢按中枢震荡处理。

## MACD 背驰辅助细则与买点定律

来自第 15 课 page 064、066-071 的配图问答。后续使用 MACD 辅助背驰时:

- MACD 柱子必须按一波来看,不能只看单日或单根柱。红/绿柱一波先伸长后缩短,关键是本波转折高度与上一波转折高度比较。
- 最有效的顶背驰常见形态: 黄白线回到 0 轴附近再上,价格创新高,但黄白线和柱峰都不创新高。底背驰反向处理。
- 若当波柱峰仍在扩张,或黄白线仍强创新高,只能说明走势正在形成中,不能硬判本级别背驰。
- 若只是黄白线仍创新高但红柱缩短,通常只能说明更小级别背驰或动能放缓。5m 黄白线大幅创新高时,即使柱子缩短,多半只是 1m 级别背驰。
- MACD 买点定律只作辅助校验: 第一类买点多在 0 轴之下的背驰中形成;第二类买点多在第一次上 0 轴后回抽确认中形成;卖点反过来。
- 同一只票可能 30m 背驰已发生而日线没有背驰。输出必须写清背驰所在级别,不能把分钟背驰直接升级为日线背驰。
- MACD 柱面积不必等全部走完才估算。若柱子开始缩短,可按已出现面积和衰减节奏估算,用于末段卖出或买入的精确定位。

## 趋势背驰、盘整买点与反抽纪律

来自缠中说禅 2006-12-11 关于趋势背驰、除权位置、跌停后强力反抽的问答。后续判断背驰和短线反抽时:

- 找背驰必须先确认是趋势。若走势没有两次以上同级别缠绕/中枢演化,就不能叫标准趋势背驰;最多只能写成 `盘整背驰` 或 `类背驰力度比较`。
- 盘整中一次缠绕后也可能有买点,但这只是盘整内部的次级别买点或回补点,不能升级为本级别一买/二买。普通小级别盘整不宜重仓参与。
- 周线/月线级别盘整底部可单独重视,因为级别大、风险位清楚、利润空间可能足够;小级别盘整买点只能按短差或试错处理。
- `女上位` 或短期均线在长期均线上方的强势状态中,每次缠绕后买入都可能赚钱,但原文只把第一次缠绕后的买点定义为第二类买点。后续多次缠绕不能重复标成本级别二买,只能写成趋势延续买点、回补点或小级别买点。
- 底部买入的核心是风险好控制。任何一买/二买/三买候选都必须绑定失效位;失效位跌破后,候选立即失败,不能用更大级别叙事安慰持有。
- 除权对 5 日线/10 日线以下的短线操作影响较小,可按不复权图处理;若分析周线、月线或大级别中枢/均线结构,应使用复权口径保持走势连续。输出中若复权影响合笔或中枢边界,必须注明使用口径。
- 跌停后的强力反抽不能当趋势买点。日线短线先看 5 日线是否能站住;更短线要下钻到 1m/5m 看反抽力度。
- 抢反抽时,上涨过程中一旦出现力度衰竭、类背驰或冲不过关键均线/中枢边界,就按短线卖点处理。不能等到均线全部下穿或大级别完全走坏后才反应。

## 形态、缺口与实际操作级别

- 实际买入级别决定退出级别。若买入点不是日线/周线第一、第二类买点,而只是 15m 箱体或 1m/5m 短线点,就不能拿日线卖点来安慰持有。
- 箱体操作按箱体边界执行: 箱顶不破先卖或降仓,回到箱底/回补点再买回;突破箱顶后再观察是否升级,不能预设一定突破。
- 三角形判断直接看高低点收敛,不看均线。三角形整理可成为第二类买点构造的一部分,但前提是前面已有本级别一买或底部结构,且回抽不破关键位。
- 三角形常出现在一段走势最后一个调整中。三角形突破后,后续走势可能进入末段加速或背驰段,要密切关注 30m/60m 或操作级别背驰。
- 缺口以日线等大级别为准。日线有缺口时,低级别自然有缺口;若低级别软件未显示缺口,只是软件口径问题。
- 缺口不必一概回补。突破性缺口、岛型反转缺口不补反而表示强势;重点防范的是盘整上涨末端的衰竭性缺口。
- 简单持仓纪律可按能力分层: 短线看 5 日线,中线看 5 周线,长线看 5 月线;未有效跌破就按对应级别持有。
- 有效跌破必须由对应级别反抽确认,不是一根 K 线跌破就成立。例如月线跌破至少要看下个月级别反抽,不能用一天反弹替代月线确认。
- 已从低位持有的仓位,高位震荡阶段重点是先卖后买降成本,不是继续追买。理想短差结果是持仓数量不变、成本不断下降。

## 小级别背驰引发大级别转折

- 小级别背驰本身不是大级别转折充分条件。
- 小级别顶背驰引发大级别向下转折的必要条件: 最后一个次级别中枢出现第三类卖点。
- 小级别底背驰引发大级别向上转折的必要条件: 最后一个次级别中枢出现第三类买点。
- 对本级别持仓,小级别背驰只能先做风险提示或部分处理;是否全退出,要看是否破坏本级别关键中枢/买点。
- 按 30m 操作时,小于 30m 的顶背驰首先至少应引发 5m 向下走势。若该 5m 下跌没有回到构成最后 30m 中枢三买的 5m 回试走势高点,通常仍属 30m 可承受回调;若跌破该高点,后续向上回抽必须先离场或大幅降仓。
- 看到 1m 顶背驰就全仓卖出,实际操作级别已经变成 1m。若主操作级别仍是 30m,只能把 1m 顶背驰当风险提示或短差卖点,并等待是否破坏 30m 关键结构。
- 资金量较大时,同一卖点不可能全仓处理。可在小级别顶背驰或最后 5m 中枢三卖处分层减仓,若关键位不破再回补;底背驰时反向使用。
- 快速赶顶股票的最后上升段可能只是 1m 趋势延伸。此时 1m 标准背驰可能快速打回延伸起点附近,不能因为级别低就忽视短线风险。
- 背驰后的回撤幅度没有固定比例。上升中段强势时,5m 背驰通常只适合打短差;日线趋势末端或延伸段中,1m 背驰也可能触发暴跌。

## 第 25 课中枢与操作级别补充

来自 page 144-159 的“吻、MACD、背驰、中枢”和操作级别问答。后续判断 MACD、中枢和低级别背驰时:

- 中枢只能在回试、回拉或来回震荡中形成,不能用离开段本身直接确认。
- 日线中枢必须由至少三段 30m 走势类型构成;30m 中枢必须由至少三段 5m 走势类型构成。更小级别走势只能递归生成直接次级别走势类型,不能越级确认父级别中枢。
- 一条直线式 30m 上涨或下跌不是日线中枢的三段构成;必须等 30m 级别至少三段走势类型完成后,才能确认日线中枢。
- 第三类买点只能在中枢之外形成;三角形内部或中枢内部的最后买点只能标为次级别买点。
- 第二类买点/卖点不能只因第一次回调/反抽不破就严格成立;必须等待次次级别背驰或双次回拉确认。工程上 `classify_bsp_label()` 若无 `sublevel_divergence` 或 `double_retrace` 证据,只能输出 `2B候选/2S候选`。
- 周线偏中线,日线/60m/30m 偏短线,5m 属超短,1m 只对 T+0 或极短短差有意义。A 股 T+1 下,1m 信号更不能写成当天可闭环的完整交易计划。
- 60m 背驰要用 60m K 线观察,不能用 5m 背驰替代;若 60m 不出背驰但迟迟不能拉起,只能写作风险观察或走势弱化,不能硬判 60m 背驰。
- 日线操作不需要盯 1m 图做结论;只有在次级别背驰、精确定位或风险反证时才下钻。
- 快速拉开末段里,1m 背驰可能触发剧烈回落,但它仍先属于低级别风险信号;是否升级,取决于本操作级别关键中枢/买点是否被破坏。
- 1m 标准背驰仍要符合黄白线回抽 0 轴后再创新高/新低而 MACD 不创新高/新低。若 5m 无背驰,1m 背驰只能按机动资金短差处理。

## 第 26 课市场风险与交易连续性前提

来自 page 168-169 的“市场风险如何回避”。后续把缠论用于交易或扫描时:

- 缠论技术控制的前提是交易品种在操作级别的可预见时间内可以连续交易。若存在停牌、长期无法交易、流动性枯竭或交易中断风险,不能只靠技术结构控制风险。
- 交易资金也必须匹配操作级别的时间长度。若资金有明确期限,应降低操作级别到期限可覆盖的范围,或不参与。
- 基本面、消息面、政策、经营、流动性等风险最终都必须反映到走势上;分析时可以记录风险源,但执行上仍以走势、买卖点、失效位和是否可交易为准。
- 停牌/无法交易类风险不属于走势内风险,应进入交易过滤或风控备注,不能用“后续有买点再走”来处理。

## 同级别分解

- 选定一个固定级别做同级别分解,可把市场机械分解成同级别上涨、下跌、盘整。
- 固定 30m 级别后,30m 上涨就是该级别牛市,30m 下跌就是该级别熊市。牛熊只是操作级别下的标签,不能用宏观牛熊叙事替代分解规则。
- 操作中只参与同级别上涨和可控盘整,回避同级别下跌。
- 同级别分解必须保持唯一性和可操作性;多义性不是随意乱分解。
- 对持仓管理,可以分仓位采用不同级别节奏,但每一部分仓位必须有自己的固定操作级别和退出规则。
- 背驰和盘整背驰是走势分段依据。第三类卖点更多是盘整结束确认,不是唯一卖出触发;上涨进入背驰区间或第二卖点时就应处理。
- 低级别信号低于当前操作级别时,只作短差或风险提示;同级别信号才可进入该操作级别的买卖决策。
- 第一卖点没走,就要在第二卖点走。等到第三卖点往往跌幅已大;卖早只是少赚,卖晚会被动。
- 小级别操作可在连续段组合出高一级别中枢后自动换挡到高一级别,但换挡必须有结构生成,不能因盈利或亏损临时改级别。
- 多级别仓位要分层: 小级别短差、中级别波段、大级别核心仓各自绑定级别、仓位和节奏,不能互相污染。

## 走势必完美

- 任意走势类型都会完成;级别越大不代表不会完美,只代表完成时间和空间更大。
- 高级别转折必然先从低级别开始,所以区间套有操作意义。
- 走势判断不能预测反弹还是反转;只看当下是否形成对应级别中枢、背驰、第三类买卖点和非背驰离开。

## 板块强弱与轮动

- 均线系统可作为级别强弱的辅助,但均线不是级别本身。
- 板块强弱可用板块内个股穿越/压制均线类别做平均,对应现代实现中的板块 RPS。
- 个股买点若没有板块强弱配合,只能降低仓位或当反弹处理。
- 最强板块的轮动节奏必须与个股中枢/买卖点结合,不能只因板块强而追高。

## 短线反弹操作

- 若判断至少有一个 30m 级别反弹,不能等 30m 中枢完全走完才处理,应按 5m 节奏分解。
- 反弹是否演变成反转不需要预测;只有第一中枢后出现三买,并形成非背驰向上,才可升级为更强持有。
- 反弹票要先看可控止损,再看目标;无法控制止损的反弹不做。

## 后续输出硬要求

个股缠论分析必须回答:

1. 当前操作级别是什么,是否临时切级别。
2. 日线、60m、30m 最近几笔人工合笔结果。
3. 各级别最近中枢的 `ZD/ZG` 与当前价位置关系。
4. 信号属于一买、二买、三买、盘整背驰、小转大候选,还是只是工具误判。
5. 若工具信号与人工结构冲突,以人工结构为准并说明冲突原因。
6. 买入/持仓/卖出必须给出失效位,不能只给方向判断。

## 深度思考: 对当前工程信号的约束

### 1B/2B/3B 不能直接等同原文一买/二买/三买

当前扫描器里的 `1B/2B/3B` 是工程标签,不是严格原文买点:

- `1B`: 多数只是低位起涨笔或小级别修复,可能对应非标准一买、盘整背驰、小转大候选或 C 观察点。没有同级别下跌趋势背驰时,不能叫标准一买。
- `2B`: 必须看是否是一买后的第一次次级别回调不破。若只是回踩不破近期低点,只能叫工程二买候选。
- `3B`: 必须满足“次级别离开中枢 + 第一次次级别回试不破 ZG”。若只有突破、没有回试,或回试级别不清,不能叫标准三买。

后续输出要优先写成:

- `工具标签: 1B/2B/3B`
- `人工结构: 标准买点 / 非标准买点 / 候选 / 失败`
- `可交易性: 入场 / 持仓 / 观察 / 不碰`

### 买点当下执行,回溯定性

缠论的本质是在不确定中找到可执行的确定性。以后分析买点必须把“当下可操作”和“事后确认”分开:

- 当下买点只要完成对应级别的结构条件,只能写成 `候选` 或 `试错`: 下跌衰竭/背驰、底分型、向上笔、回踩不破、离开中枢后的首次回试等。
- 回溯确认不是说买点当时不存在,而是用后续结构给过去的候选定性: 不破前低确认二买,突破前高确认前低有效,离开中枢后回踩不回中枢确认三买。
- 一买确定性最低但成本最低,只能小仓试错;二买确认度更高,适合加仓;三买确定性最高,适合趋势仓,但成本更高。
- 每一个候选买点必须绑定失效位: 一买跌破候选低点失败;二买跌破一买低点失败;三买跌回中枢或跌破 `ZG` 失败。
- 输出时禁止只说“回溯确认后买点已经过去”。正确表达是: 原低点已过去,但二买/三买是后续更高确定性的执行点。

### A 股 T+1 下的当下执行

A 股交易默认为 T+1。缠论候选买点可以用于当下试错,但必须先折算 T+1 锁仓风险:

- 今日买入的新仓 / 加仓今日不能卖出;“跌破就走”只能用于已有可卖仓位,对新买股份必须写成“次一交易日可执行止损”。
- 候选试错不是原文严格买点,只能写 `执行试错仓`,默认 `loose 2-3%`;不能升级成 `medium` 或 `strict`。
- 试错只允许发生在靠近失效位、靠近中枢下沿、或回踩确认位;中枢中部、上沿追高、距离失效位过远时,候选也不做。
- 早盘和盘中试错要更轻,因为当天不能卖出新买股份;临近收盘且结构仍有效时,才允许比盘中更接近计划仓位。
- 若候选收益空间不足以覆盖次日低开、跌停、滑点和失效位风险,不做。

### 当下性优先于预测

原文强调走势当下性。后续分析必须避免:

- “应该会反弹”“有可能主升”这类无结构预测。
- 因题材、板块或成本位而预设走势方向。
- 用未完成的中枢、未完成的笔去提前确认买卖点。
- 某周期图找不到符合定义的分型时硬凑分型;若要看得更细,可换小级别图当显微镜,但这不等于改变操作级别。

应改成:

- 当前已经完成了什么结构。
- 下一步只有哪几种完全分类。
- 每一种分类对应哪个点位触发和失效。
- 当下与事后应使用同一套客观定义。若事后回看与当下判断差异大,通常是当下没有严格处理包含、分型、笔或线段。

### 中枢震荡要单独归类

很多误判来自把中枢震荡中的低点当一买、把高点突破当三买。以后必须先判断是否还在中枢震荡:

- 未形成第三类买卖点前,本级别震荡仍在延续。
- 震荡内部的买卖只属于次级别,不能上升为本级别买卖点。
- 可用中枢中轴 `Z=(ZD+ZG)/2` 与各次级震荡中轴 `Zn` 判断偏强/偏弱。
- `Zn` 持续在 Z 上方但不突破 ZG,要防诱多;持续在 Z 下方但不跌破 ZD,要防诱空。

### 小转大只给必要条件,不做充分确认

后续遇到“5m/30m 背驰带动日线转强”时,必须写成候选,不能直接确认:

- 小级别底背驰后,若最后一个次级别中枢没有三买,不能确认大级别向上转折。
- 即使出现三买,也只是必要条件,仍需观察是否突破大级别关键中枢边界。
- 对持仓可先减压或试错,但不可满仓押注“小转大一定成立”。

### 同级别分解决定仓位纪律

同一只股票可有多套仓位,但每套仓位必须绑定自己的操作级别:

- 日线仓: 不因 1m/5m 波动频繁买卖,只用小级别做精确定位或风险提示。
- 30m 仓: 30m 下跌段或 30m 三卖出现时必须退出/降仓。
- 5m/1m 试错仓: 失败必须快出,不能升级成日线仓来安慰持有。

持仓诊断时必须说明用户当前仓位更像哪一级别仓位。如果真实成本远高于当前结构买点,不能用旧系统买点替代当前风控。

### 板块强弱只能决定赔率和仓位

板块强弱指标/RPS 对应原文 106 课的轮动思想,但不是买点本身:

- 强板块 + 标准买点: 可提高优先级和仓位。
- 强板块 + 无买点: 仍不能追。
- 弱板块 + 买点: 降仓或只做反弹。
- 个股显著弱于板块: 视为掉队,即使板块强也不能放宽结构要求。

### 通读后的工程修正(2026-06-04 用户修订:T+1 试错折扣)

1. **统一以原文严格为唯一买卖点定性标准**,不把"工程候选 / 工程标签"写成严格 1B/2B/3B。
2. **扫描脚本(chanlun_low_start v0.4.3 / sector_jump_v1 / stock_deep_analysis 的 ZigZag 笔)输出的 1B/2B/3B 是「扫描候选」**,只表示"疑似买点",必须经人工合笔和原文严格校验后才决定能否确认。
3. **1B 必须满足三项**:本级别下跌趋势成立 + c 段同级别背驰 + 反弹笔成立。**任一缺失 = 不确认严格 1B**。
4. **3B 必须满足三项**:离开 ZG + 第一次次级别回试 + 不破 ZG。**任一缺失 = 不确认严格 3B**。
5. **未达原文严格 = 不能给确认仓 / 趋势仓**,只能写观察、等待触发,或在靠近失效位且 T+1 风险可承受时写 `执行试错仓 loose 2-3%`。
6. **A 股 T+1 必须折扣执行**:今日买入不能今日卖出;所有候选试错都必须给次一交易日止损位,距离失效位过远或中枢中部时不试错。
