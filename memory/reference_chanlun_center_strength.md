# Chanlun Center Strength Reference

Date: 2026-06-21

## 适用场景

用于分析本级别中枢内部震荡、离开中枢、回试中枢、跌破后回补、突破后真假强弱、第三类买卖点质量和持仓 / 做 T 边界。

凡是用户问“中枢力度”“中枢强弱”“离开力度”“回试力度”“是否强三买 / 弱三卖”“中枢上移 / 下移”时,必须使用本规则。

## 核心定义

中枢力度不是一个单独买卖点,而是围绕同一个本级别中枢的同级别 / 直接次级别走势段强弱比较。

它回答的是:

```text
价格围绕该中枢,是更容易向上离开、向下离开,还是继续震荡。
```

中枢力度必须先固定:

- 操作级别。
- 中枢边界 `ZD / ZG` 和中轴 `Z = (ZD + ZG) / 2`。
- 构成中枢的三段直接次级别走势。
- 要比较的两段或多段走势是否同级别、同方向、围绕同一中枢。

力度 evidence 包括:

- 价格:离开幅度、收盘是否站稳中枢外、是否重新回到中枢。
- 时间:离开 / 回试用了多少 K,是否急拉急跌后立刻回落。
- MACD:同级别或直接次级别红绿柱面积、柱峰、DIF / DEA 是否扩张或收敛。
- 量能:离开放量还是缩量,回试缩量还是放量砸回。
- 位置:震荡中轴 `Zn` 相对中枢中轴 `Z` 的上下偏移。
- 结构:是否形成直接次级别离开段 + 直接次级别回试段,以及回试是否不回中枢。

## 判定步骤

1. 先定操作级别,例如 `30m` 或 `日线`;不能把不同级别混在一起比较。
2. 画出本级别中枢 `ZD / ZG / Z`,确认三段直接次级别走势有共同重叠。
3. 判断当前价格位置:中枢内、ZG 上方、ZD 下方、回试中、回抽中。
4. 若仍在中枢内,比较各次级震荡中轴 `Zn` 与 `Z`: `Zn` 多数在 `Z` 上方为中枢内偏强,多数在 `Z` 下方为中枢内偏弱,但都不是本级别买卖点。
5. 若向上离开 `ZG`,比较离开段力度:放量、MACD 扩张、收盘连续在 `ZG` 上方,为强离开 evidence;缩量、冲高回落、快速跌回中枢,为弱离开 / 假突破 evidence。
6. 若向下离开 `ZD`,镜像比较:放量下跌、MACD 绿柱扩张、收盘连续在 `ZD` 下方,为强下离开 / 风险 evidence;下破后力度衰竭并快速回中枢,为跌破后回补 evidence,不是新增买点。
7. 若离开后第一次回试 / 回抽:回试缩量、不回中枢、随后再创新高,支持 3B 质量提高;回试放量、跌回中枢、跌破中轴甚至跌破 `ZD`,说明离开失败,按中枢震荡或转弱处理。
8. 输出结论时只给 `强 / 中性 / 弱 / 衰竭 / 假突破风险` evidence;是否构成 1B/2B/3B/1S/2S/3S,仍回到原文严格买卖点规则。

## 输出要求

分析中枢力度必须给表:

```text
级别:
中枢:ZD / ZG / Z
当前位置:
比较段:
价格力度:
MACD力度:
量能力度:
回试/回抽:
力度结论:
买卖点结论:
执行结论:
失效位:
```

禁止只写:

```text
中枢很强,可以买。
突破中枢,三买成立。
MACD 变红,中枢力度强。
```

必须写成:

```text
30m 中枢 29.94-30.86,中轴 30.40。
本次向上离开 ZG 后,5m 回试缩量且低点仍在 ZG 上方,支持 30m 3B 质量提高。
但严格 30m 3B 仍需确认这是离开后的第一次直接次级别回试,并且 5m 回试走势已完成。
```

## 正例

```text
本级别:30m
中枢:ZD 29.94 / ZG 30.86 / Z 30.40
当前位置:ZG 上方第一次回试
比较段:5m 离开段 vs 5m 回试段
价格力度:离开段连续收在 ZG 上方,回试低点 30.91 未回中枢
MACD力度:离开段红柱扩张,回试段绿柱面积小
量能力度:离开放量,回试缩量
力度结论:向上离开强,回试健康
买卖点结论:30m 3B 标准候选;若确认首次直接次级别回试完成,可升级严格校验
执行结论:只在靠近回试低点且 T+1 风险可控时执行,追高不做
```

## 反例

```text
日线中枢上方涨停,所以日线三买成立。
```

错因:只有向上离开,没有确认日线直接次级别的第一次回试不回中枢。

```text
5m MACD 红柱很强,所以 30m 中枢力度强。
```

错因:没有说明 5m 段是否是 30m 中枢的直接次级别离开段或回试段;单独 MACD 不能替代结构。

```text
中枢内价格多次在上半区,直接买。
```

错因:中枢内偏强只是 evidence,未离开 `ZG` 且未形成三买前,本级别仍是中枢震荡。

## 优先级

本规则是 `memory/reference_chanlun_108_core_rules.md` 的 evidence 补充,不改变原文严格买卖点定义。若“中枢力度强”与严格买卖点条件冲突,以严格买卖点条件为准。

中枢力度可以提高或降低候选质量、仓位档和等待路径,但不能单独把候选升级为严格 1B/2B/3B,也不能推翻 M、止损、破位、板块、机构资金和业绩硬信号。
