# sector_jump_v1 — 板块跳升驱动公式 (A股 实战版)

## 核心定位

`sector_jump_v1` 是基于 §7.2 板块 RPS + §M 大盘 + §L 龙头识别的纯量化扫描公式,
解决 chanlun_low_start v0.4.3 漏过电力央企等板块轮动龙头的盲区。

它**不是**贴底公式,而是 **板块从冷转热 + 个股领涨** 的双信号公式。

## 入场公式

```
板块: sw_rps5 ≥ 90 + 5 日跳升 ≥ 20 (从冷转热)
个股: RPS5 ≥ 70 (板块成员里非掉队腰部领涨)
价格: ≥ 5 元
缠论: ZigZag 5% pivot=L + 笔起距今 ≥ 3 日 + close > 笔起 × 1.005 + 非顶分型
A股 约束: 当日非涨停板 (买不到)
月份: 跳过 BEAR 月份 (2025-03 / 2025-11 / 2026-02)
```

## 完整策略 (回测验证 +5.27% / Sharpe 0.307)

### 止损 (双层)

```
技术止损 (优先): 跌破 bi_low (缠论笔起, 通常 -3%~-5%)
                → 立即清仓, 减少 -8% 大败次数

硬止损 (兜底):   跌破 入场价 × 0.92 (-8%)
                → 必须清仓 (§5.3 铁律不可松)

⚠ A股 跌停顺延规则:
  当日触发止损但跌停 (close ≤ 前 close × 0.903) → 卖不出, 次日开盘卖
  实际亏损可能扩大到 -15%~-40% (频率 < 2%)
```

### 止盈 (三段)

```
T1 +15% → 减仓 50%      (锁第一段收益, ~30% 票触发)
T2 +25% → 再减仓 25%    (保护浮盈)
T3 +40% → 全清剩余      (主升龙头才到这里, ~7% 触发率)
```

### 加仓 v2 (浮盈回踩 ma5)

```
触发条件 (都满足):
  1. 浮盈 ≥ +5% (相对加权平均成本)
  2. 当日 low ≤ ma5 ≤ high (回踩 ma5)
  3. close ≥ open (收阳确认)

加仓量: +50% 原仓位 (单仓 4% → 加仓后 6%)

⚠ A股 约束:
  - 当日涨停 (close ≥ 前 close × 1.097): 加仓被拒, 买不到
  - 加仓当日不能卖 (T+1)
  - 单股总仓位 ≤ 加仓后上限 6%
```

### 时间上限

```
30 个交易日 (而非机械 20 日)
期间不机械退出, 只看价格触发
```

## 仓位 (DIST 状态默认)

```
A1 主仓:    4% × 总资金 (而非理论 5-8%, 防跌停跳空)
A0 试错:    2.5%
单板块:    持仓 ≤ 3 只 (避免集中度风险)
加仓后:    单仓最大 6%
总仓位:    ≤ 50% (DIST), 80% (BULL), 20% (BEAR)
```

## 实盘脚本

```

加仓 触发:  浮盈 ≥ +5% 后回踩 ma5 不破 + 收阳

```

实盘工作流:

4. **DIST 状态** 单仓全降一档, 单板块 ≤ 3 只
