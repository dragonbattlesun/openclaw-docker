## 总原则

### 先缠论结构,后经典形态

经典图表形态不能替代缠论的级别、走势类型、中枢和买卖点。使用顺序固定为:

```text
操作级别 -> 分型/笔/线段 -> 中枢 ZD/ZG -> 走势类型 -> 买卖点身份 -> 经典形态辅助解释
```

执行口径:

1. 形态边界不是中枢边界。只有由至少三个连续直接次级别走势类型重叠生成的区间,才能写 `ZD/ZG`。
2. 经典“突破”只等同于离开段 evidence,不能直接写成第三类买点。严格 3B 必须有直接次级别向上离开中枢,再第一次直接次级别回试不回中枢。
3. 经典“跌破”只等同于向下离开 evidence,不能直接写成第三类卖点。严格 3S 必须有直接次级别向下离开中枢,再第一次直接次级别回抽不回中枢。
4. 头肩、双重顶底、楔形、缺口、趋势线和支撑阻力只能辅助判断力度衰竭、假突破、离开段质量或风险,不能新增一买/二买/三买/一卖/二卖/三卖。
5. 若经典形态看涨但缠论无买点,交易动作仍写 `不碰`;可以另列 `观察 / 候选`,等待缠论买点或直接次级别确认。

### 形态必须先分级别

同一个图形在日线、周线、月线上的含义不同。输出时必须写清:

```text
级别:
前置趋势:
形态边界:
量能:
突破:
回抽/反扑:
最小目标:
失效条件:
```

没有级别、前置趋势和突破状态,只能写“形态候选”,不能写确认。

### 突破确认

经典图表形态的确认通常依赖“决定性突破”:

1. 收盘价有效穿越形态边界或趋势线。
2. 经典口径常用约 `3%` 的穿透幅度作有效性参考。
3. 向上突破通常必须有成交量明显放大配合。
4. 向下跌破不一定需要放量,但放量跌破会加强有效性。
5. 突破后反扑/回抽到原边界很常见,只要不重新回到形态内部并破坏边界,不能简单否定突破。

A 股实盘中不能机械套 `3%`:涨跌停、T+1、流动性和个股波动率要一起看。但也不能把轻微越线写成确认突破。

### 量能是形态质量过滤器

- 持续/整理形态内部通常应量能递减。
- 上破边界时量能应显著增加。
- 底部形态的上破比顶部形态的下破更依赖放量。
- 若形态内部量能一直很高且不规则,要警惕这不是整理,而是反转或派发。

### 最小目标不是盈利承诺

头肩、矩形、旗形、三角形等形态都有测量规则,但测量值只表示“最小或理论运动空间”。实际交易仍要看:

- 大盘 M 是否支持。
- 板块是否共振。
- 个股是否有缠论买点/卖点或至少结构候选。
- 止损空间与目标空间是否满足 `收益 / 风险 >= 2:1`。
- A 股 T+1 下当日新买是否可能立刻失效。

## 具体判定规则

### 对称三角形

判定步骤:

1. 必须先有前置趋势。
2. 至少看到四个细小趋势反转点:上升后形成一个顶部、一个底部、一个更低顶部、一个更高底部;下跌后反向同理。
3. 只有第二个底部/顶部远离后,才能画出两条收敛边界线。
4. 形态内部量能应逐渐萎缩。
5. 价格突破前,方向没有结论;不要提前押方向。

输出要求:

- 写“对称三角形候选/确认”,不要只写“三角整理”。
- 上破必须看成交量显著增加。
- 下破不一定要求放量,但若跌破后马上回到形态内,要按假突破处理。
- 靠近顶点才突破时可靠性下降;顶点附近假动作更多。

### 直角三角形

上升三角形:

- 上边界近似水平,代表固定价位的供给。
- 下边界逐步抬高,代表需求增强。
- 只有放量突破水平上边界才确认。
- 若第三/第四次回落跌破上升需求线,形态可能转为矩形或失败。

下降三角形:

- 下边界近似水平,代表市场相信的支撑。
- 上边界逐步压低,代表反弹越来越弱。
- 跌破水平支撑后,原持有者信心容易被打破,下跌可加速。
- 关于对称三角形的多数突破规则也适用直角三角形。

### 矩形

矩形是水平交易区间,描述供需双方势均力敌。

判定:

1. 顶部和底部都能用水平边界界定。
2. 区间延伸时量能逐渐减少。
3. 突破边界前不预设方向。
4. 矩形更常是持续形态,但底部反转矩形也常见。

确认:

- 有效突破上沿或下沿。
- 突破后 3 天到 3 周内反扑/回抽边界较常见。
- 最小目标可用矩形高度从突破点投射。

反例:

- 区间内量能越来越大且混乱,不能当健康矩形。
- 未突破就按突破方向交易,只是猜方向。

### 旗形与三角旗

成立条件:

1. 必须发生在快速、直线式、大规模运动之后。
2. 整理时间短,通常数天到三周;三周以上警惕,四周内必须离开。
3. 形成过程中成交量显著减少。
4. 突破时沿原趋势方向恢复动能。

测量:

- 从前一巩固/反转形态突破处或重要趋势线突破处作为旗杆起点。
- 量到旗形开始处的距离,再从旗形突破点沿原趋势投射。
- 这是最小目标,不是止盈承诺。

失效:

- 形态内部量能不缩反增。
- 整理时间拖太久。
- 突破方向与前趋势相反且伴随有效量价,应按反转风险处理。

### 楔形

楔形由两条同向倾斜且相交的边界构成,不同于三角形的一上一下收敛。

上升楔形:

- 两条边界都上倾,下边界通常更陡。
- 价格仍在上升,但每一波上攻更弱,需求衰退。
- 跌破下边界后,常常快速回吐楔形内涨幅。
- 它可以出现在既有上升趋势的末端,也可以出现在下跌趋势中的反弹段。

下降楔形:

- 两条边界都下倾,上边界通常更陡。
- 完成后常转向上行或至少进入更大震荡。
- 若有一条边界接近水平,要优先检查是否是直角三角形。

输出时不要把“上升楔形”误读为更强的上升趋势。楔形的关键是动能衰退,不是方向本身。

### 头肩形态

头肩顶:

1. 左肩通常伴随强成交。
2. 头部交易量可能高于、等于或低于左肩,不能只凭这一点否定。
3. 右肩反弹时量能明显不能恢复,是真正警告。
4. 跌破颈线约 3% 才确认。
5. 最小跌幅:头部顶点到颈线的垂直距离,从颈线跌破点向下投射。

头肩底:

1. 头部阶段和右侧反弹量能应逐步增强。
2. 放量突破颈线是必要确认。
3. 缩量突破颈线不能当决定性确认,可能仍需继续筑底。
4. 头肩底通常比头肩顶更长、更平缓、更圆。

复合头肩:

- 可有多个肩部,对称性更强。
- 底部复合头肩比顶部更常见。
- 形态可靠性不低,但首段运动常在最小目标附近出现强反弹/回调,不要高估持续力度。

头肩持续形态:

- 形状与普通反转头肩类似,但相对前趋势方向是“倒置”的持续整理。
- 交易量结构不同于反转头肩。
- 测量目标不如普通反转头肩可靠,需要结合其他证据。

### 双重顶与双重底

双重顶成立前提:

1. 两个峰位大致相同,可用约 3% 容差观察。
2. 两峰时间间隔越长越可靠;多数真实双重顶间隔 2-3 个月或更长。
3. 中间回落越深越圆滑,第二次上攻量能越弱,顶部风险越高。
4. 只有跌破两峰之间谷底支撑,才最终确认。

双重顶确认后:

- 往往是主要反转,不要低估。
- 最小跌幅可按峰顶到谷底的高度向下估算。
- 跌破后反扑谷底区域很常见。

双重底:

- 第二底是对支撑的测试。
- 第二底后反弹应有量能增加。
- 只有放量突破颈线,才证明测试通过。

### 三重顶与三重底

三重顶:

- 第三个峰形成时只能警惕,不能确认。
- 必须跌破谷底/颈线,才确认趋势反转。

三重底:

- 是三重顶的倒置。
- 必须向上突破颈线,且最好量能放大。

### 扩散形态

扩散形态表现为波动越来越大,高点抬高、低点降低,通常伴随高而混乱的成交量。

关键规则:

1. 多见于长期牛市末期或最后阶段。
2. 与矩形的“有控制的矛盾”相反,扩散代表公众情绪激动、交易混乱。
3. 需要至少第三个更高顶部后,再跌破前一个重要低点,才构成强反转证据。
4. 跌破后常出现反扑,可回抽前一跌幅的 1/2 到 2/3。
5. 不要因为扩散区内仍创新高就追买;高点创新但低点也降低,是结构失控。

### 圆底、圆顶与扇形

圆底/碟形底:

- 多见于低价股。
- 形成时间长,常需数月。
- 供应逐渐枯竭,趋势逐步走平,底部成交极少。
- 右侧需求恢复后,价格和成交量一起加速。
- 后续上升常慢且被反复打断,需要耐心。

圆顶:

- 比圆底更少见,且量能信号不如圆底清楚。
- 重点观察价格达顶后的小范围波动是否由强转弱。

扇形/连续碟形:

- 某些低价且交易活跃的股票,会以一系列碟形/扇形推进长期主升。
- 真正机会往往在平静的碟形底部和边缘高点确认处,不是交易突然活跃后的追买。
- 只要图表未显示明确行为改变,可假设扇形特性继续;一旦分离迹象出现,要重新评估。

### 缺口

缺口不是神秘信号。不是所有缺口都必须回补;缺口是否有意义取决于位置、宽度、量能和前后形态。

普通/区域缺口:

- 出现在交易区域或巩固形态内部。
- 多数会在形态完成前回补。
- 矩形或对称三角形内部反复出现普通缺口,更支持“这是持续整理”而不是反转。

突破缺口:

- 出现在形态完成、水平边界突破或重要趋势线突破时。
- 常见于上升三角形、矩形、趋势线有效突破。
- 一般不应机械等待回补;若回抽,原边界与缺口区域常成为支撑/阻力。

持续/逃逸缺口:

- 出现在快速直线运动中段。
- 可作为测量缺口,大致标示该段运动中点。
- 如果前一形态的目标还远未完成,更可能是持续缺口。
- 第一个快速运动缺口通常优先按持续缺口观察。

衰竭缺口:

- 常出现在运动末端,前一形态目标已接近或完成。
- 缺口宽、量能异常高,常伴随反转日。
- 往往 2-5 天内快速回补。
- 它不必然等于主要趋势反转,但通常足以结束原方向短线头寸。

岛形反转:

- 两个方向相反的缺口把一段价格区间孤立出来。
- 可出现在重要趋势转折点,也可出现在局部极点。
- 第二个反向缺口完成后,短线应迅速尊重反转风险。

指数缺口:

- 指数缺口比个股缺口少见,但技术意义大体相同。
- 不要赋予指数缺口额外神秘能力;仍按形态位置和量能判断。

### 趋势线

画线:

- 上升趋势线连接回落低点。
- 下降趋势线连接反弹高点。
- 两点可画临时线,第三次接触且不破才显著提高有效性。

有效性:

1. 被测试次数越多,意义越大。
2. 未被跌破/突破的时间越长,意义越大。
3. 斜率越平缓、越接近水平,跌破意义越大。
4. 过陡趋势线容易被小整理打穿,技术价值较低。
5. 中长期趋势优先用半对数图观察。

趋势线突破:

- 可用约 3% 穿透作为有效参考。
- 若穿透不足 3%,但伴随显著放量并收在日内低位附近,也可视作强烈警告。
- 若只是盘中虚破、收盘收回,成熟趋势线不应轻易重画。
- 如果一条线只由早期两个点构成,第三点未能触及且市场形成新的有效低点/高点,可以重画。

双重趋势线:

- 有些趋势更适合用内外两条近似平行趋势线描述。
- 内线跌破是警告,外线决定性跌破才更像趋势结束。
- 反弹回不到内线,说明趋势接近尾声。

### 支撑与阻力

支撑/阻力不是线条本身,而是过去大量交易造成的供需记忆。

判断强度:

1. 该价格区过去成交量越大,支撑/阻力越强。
2. 距离当前越近,影响通常越直接。
3. 价格离开该区域越远、持仓浮亏/浮盈越大,回到该区域时反应越明显。
4. 区域持续时间越长,影响越大。
5. 整数价位、历史活跃价位、前重要底/顶都可能形成心理支撑/阻力。

角色转换:

- 支撑跌破后转为阻力。
- 阻力突破后转为支撑。
- 头肩颈线、矩形边界、三角形边界、突破缺口边缘都适用角色转换。

输出时必须写清“支撑/阻力来自哪里”:前高、前低、成交密集区、形态边界、缺口边缘、趋势线或整数位。不能只写“上方压力大”。

## 与项目交易框架的衔接

### 形态只属于 T 维度

经典图表形态在项目中只属于 `T:技术结构` 的一部分。它不能覆盖:

- M 大盘状态。
- 止损/破位/ma200。
- 板块强弱。
- 机构资金。
- 业绩。
- 缠论原文严格买卖点。

### 三方共振用法

强交易信号必须尽量满足:

```text
缠论结构买点/卖点或至少人工结构候选
+ 经典形态与离开/回试/背驰路径一致
+ CANSLIM / 板块 / M 不否决
+ 赔率 >= 2:1
+ A 股 T+1 风险可控
```

如果只有形态突破,但缺少缠论买点和 CANSLIM 支撑,交易动作仍写 `不碰`;可另列为 `观察 / 候选`,不能直接升级为入场。

### 输出模板

```text
形态:
级别:
前置趋势:
边界:
量能:
突破状态:

最小目标:
失效条件:
与缠论关系:
与 CANSLIM / 板块关系:

```

## 正例

```text
形态:上升三角形确认
级别:日线
前置趋势:右侧上升后的横向整理
边界:上沿 12.30,下沿低点逐步抬高
量能:整理中缩量,突破日量比 2.1
突破状态:收盘 12.75,高于上沿约 3.7%
最小目标:三角形高度约 1.20,理论 T1 13.50 附近
失效条件:放量跌回 12.30 以下或回踩上沿失败
当前交易含义:形态 T 通过;仍需 M、板块、缠论和赔率确认
```

## 反例

```text
形态:对称三角形未确认
问题:
- 只有两个高点和一个低点,不足四个反转点。
- 价格仍在三角形内部,方向未选择。
- 量能没有收缩,反而持续放大。
结论:
只能写“疑似收敛震荡”,不能写“突破前买点”。
```
