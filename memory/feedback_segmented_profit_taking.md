# 分段止盈优于单一止盈

## 教训

A股 主升节奏快 (5-10 日完成), 主升后段震荡常把浮盈吞掉.
单一 +40% 止盈触发率仅 4-7%, 96% 的票被时间到期裹挟.

## 验证数据 (sector_jump_v1 + 14 月回测)

```
单一 +40% 止盈:        平均 PnL +3.78%, Sharpe 0.334
分段 +15%/+25%/+40%:   平均 PnL +4.14%, Sharpe 0.408 (+22%)

胜率: 58.6% → 55.5% (略降, 因为减仓后剩余可能跌)
最大回撤: -8% → -8% (不变)
```

## How: 分段止盈规则

```
T1 +15% → 减仓 50%      (锁第一段收益, A1 standard early 标准 T1)
T2 +25% → 再减 25%      (保护浮盈)
T3 +40% → 全清剩余 25%  (主升龙头才到这里, ~7% 触发率)
```

### 与 §11 仓位档位对照

```
原 CLAUDE.md §11:
  A1 standard early:  T1 +10%, T2 +20%
  
修订 (验证后):
  A1: T1 +15% (半仓), T2 +25% (再减), T3 +40% (全清)
  仍兼容 §11 总仓位规则
```

## Why work

```
A股 龙头主升:
  D+0~D+10: +20%~+30% (主升)
  D+10~D+20: +5%~+15% (震荡, 浮盈被吞概率高)
  D+20+:    回调或下跌

单一 +40% 止盈:
  90%+ 的票 D+10 时浮盈 +15%~+30%, 但策略不减仓
  → D+20 时浮盈降到 +5% 才被时间止损
  → 单笔实际收益 +5% (非 +20%)

分段止盈:
  D+10 时 +15%, 减半仓, 锁定 7.5% 现金
  D+15 时 +25%, 再减 25%, 锁定 6.25% 现金
  剩余 25% 持仓, 即使后续 -10%, 总 PnL = 7.5% + 6.25% + 25%×(-10%) 
                                          = 11.25% (而非时间到期 +5%)
```

## How to apply

### 实盘管理

```
入场后每日盯盘:
  检查 1: 当日 high ≥ 入场 × 1.15? → 触发 T1, 减半仓
  检查 2: 当日 high ≥ 入场 × 1.25 且已减 T1? → 触发 T2, 再减 25%
  检查 3: 当日 high ≥ 入场 × 1.40? → 触发 T3, 全清

A股 卖出注意:
  T1/T2 触发 → 用 14:30 集合竞价附近 limit order 卖
  T3 触发 → 立即市价卖 (主升龙头一旦回头很快)
  跌停板顺延次日开盘卖
```

### 与加仓 v2 配合

```
持仓 + 浮盈 ≥ +5%:
  - 如果未触发 T1, 仍可在 ma5 回踩处加仓 +50%
  - 加仓后总仓位 ≤ 主仓 × 1.5
  - 触发 T1 时, 减仓 50% 是按当前总仓 50% 算 (含加仓)

复合效应:
  原仓 4% + 加仓 +2% = 总仓 6%
  T1 触发 → 减仓 3%, 剩余 3% 继续跑到 T2
  T2 触发 → 再减 0.75%, 剩余 2.25% 跑 T3
```
