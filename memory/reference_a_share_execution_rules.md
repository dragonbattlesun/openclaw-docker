# A 股执行约束规则

Date: 2026-06-21

## 适用场景

- A 股买点、卖点、止损、加仓、减仓、做 T 和持仓诊断。
- 候选试错仓、缠论买卖点、CANSLIM strict / medium / early / loose 执行。
- 创业板、科创板、北交所等非主板标的的风险提示。

## 核心定义

A 股执行不是理论 K 线执行。默认约束包括:

- T+1: 今日买入股份今日不能卖出。
- 涨跌停: 涨停可能买不到,跌停可能卖不出。
- 滑点、手续费、印花税: 回测理论收益必须折扣。
- 非主板权限和涨跌幅: 创业板 / 科创板通常 20%,北交所通常 30%,波动和权限风险更高。

## 判定步骤

1. 先判断是否已有可卖股份;没有可卖股份时,不能设计同日卖出动作。
2. 新开仓 / 加仓时,止损写成次一交易日可执行止损。
3. 做 T 只针对已有可卖股份;当天新买股份锁定到下一交易日。
4. 若失效位可能在买入当天被打穿,且当天无法卖出,降低仓位或放弃。
5. 候选试错只允许靠近失效位 / 中枢下沿 / 回踩确认位执行。
6. 当前价在中枢中部、上沿追高位或距离失效位过远时,写"等待,不试错"。
7. 非主板标的必须额外标注权限和 20%/30% 涨跌停波动风险。

## 输出要求

开头 5 行结论中的止损和仓位必须体现 T+1:

```text
动作:
仓位:
止损: 次一交易日跌破 X 执行 / 盘中只减已有可卖股份
目标:
有效期:
```

做 T 必须写:

```text
操作级别:
可动用股份/仓位:
卖出触发:
回补触发:
失效位:
T+1 约束:
```

## 仓位口径

- 候选试错仓默认 `loose 2-3%`。
- 只有临近收盘、结构已完成、且离失效位足够近,才可提高到 `A0 early 3-5%`。
- 原文严格 1B:结构完成后的左侧,仍按 early / 试错处理。
- 原文严格 2B:右侧确认,可按 medium 执行。
- 原文严格 3B:趋势确认,可按 strict 执行。
- DISTRIBUTION / 跌停跳空风险高时,仓位自动降一档。

## 正例

- `今日若试错,仅 loose 2-3%;新买股份 T+1 锁定,跌破 18.20 的止损次一交易日执行。`
- `做 T 只能卖已有 3000 股,回补后新增股份今日不能再卖;若回补后跌破中枢下沿,次一交易日止损。`
- `当前位置在 30m 中枢中部,离失效位过远;等待,不试错。`

## 反例

- `今天买,跌破就马上卖`。错在忽略 T+1。
- `做 T:上午买,下午冲高卖`。错在当天新买股份不能卖。
- `中枢上沿追高试错,跌破下沿止损`。错在止损距离过远且 T+1 隔夜风险不可控。

## 优先级

本规则优先于任何理论买点、工具候选、历史系统信号或回测最优参数。若结构买点和执行约束冲突,执行约束优先,仓位降档或放弃。
